Descripción del Proyecto

Este proyecto consiste en la creación de una página web estática desplegada automáticamente mediante un proceso de Integración Continua (CI). Cada vez que se realiza un cambio en el repositorio de GitHub y se ejecuta un push, GitHub Actions automatiza el despliegue de la aplicación en Surge.sh, permitiendo publicar actualizaciones sin intervención manual.

Tecnologías Utilizadas

HTML5

Utilizado para crear la estructura y contenido de la página web.

CSS3

Utilizado para diseñar la interfaz, aplicar estilos y crear la temática visual en tonos morados.

Git y GitHub

Utilizados para el control de versiones y almacenamiento del código fuente del proyecto.

GitHub Actions

Herramienta de automatización que ejecuta el proceso de despliegue cada vez que se realizan cambios en el repositorio.

Surge.sh

Plataforma de alojamiento web estático utilizada para publicar automáticamente la página en Internet.
